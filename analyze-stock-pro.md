## 一、数据采集层（来自DSA）

分析一只股票前，需要获取以下数据维度：

| 维度 | 数据来源 | 用途 |
|------|---------|------|
| **行情** | 最新价、开盘、最高、最低、昨收、成交量、成交额、换手率 | V1量价分析起点 |
| **K线** | 近60-120个交易日的日线数据（开高低收量额） | 计算均线/量价阶段 |
| **技术指标** | MA5/10/20/60、MACD、RSI(6/12/24)、成交量5日均量 | 量化趋势状态 |
| **筹码** | 筹码集中度、主力资金流向（可选） | 辅助判断主力意图 |
| **板块** | 所属行业板块、板块指数走势 | V2主线判断 |
| **新闻** | 近期相关公告、舆情（可选） | 催化/风险因素 |

## 二、量化分析引擎（来自DSA + 调参）

### 趋势判断

```
均线状态 (MA5/MA10/MA20 排列)：
  STRONG_BULL  = MA5 > MA10 > MA20 且间距在扩大 → 🟢 强势多头
  BULL         = MA5 > MA10 > MA20               → 🟢 多头排列
  WEAK_BULL    = MA5 > MA10 但 MA10 < MA20       → 🟡 弱势多头
  CONSOLIDATION = 均线缠绕                        → 🟡 盘整
  BEAR         = MA5 < MA10 < MA20               → 🔴 空头排列
  STRONG_BEAR  = MA5 < MA10 < MA20 且间距扩大     → 🔴 强势空头
```

### 乖离率（防追高）

```
BIAS_MA5 = (现价 - MA5) / MA5 × 100%
  0% ~ 3%   → 🟢 正常，可关注
  3% ~ 5%   → 🟡 偏高，等回调（来自向阳体系：不追高）
  5% ~ 8%   → ⚠️ 明显偏高，不宜买入
  > 8%      → 🔴 严重偏离，警惕回调
```

### 量能分析

```
量比 = 当日成交量 / 近5日均量
  
  量比 > 1.5   → 🔴 放量
  量比 0.7~1.5 → 🟢 正常/温和
  量比 < 0.7   → 🟢 缩量

量价配合：
  放量 + 上涨   → 资金进场 ✅（向阳V1：参与）
  放量 + 下跌   → 主力出货/恐慌（向阳V1：回避）
  缩量 + 上涨   → 惜售（向阳V1：持有）
  缩量 + 下跌   → 跌不动了（向阳V1：关注）
```

### MACD 状态

```
零轴上金叉    → 🟢 强势买入信号
零轴下金叉    → 🟡 反弹信号，需确认
死叉（零轴上） → ⚠️ 上攻动能衰竭
死叉（零轴下） → 🔴 继续弱势
```

### RSI 状态

```
RSI(6) > 70  → 🔴 超买区，不宜追
RSI(6) 50-70 → 🟢 强势区，可持有
RSI(6) 30-50 → 🟡 中性偏弱，观望
RSI(6) < 30  → 🔴 超卖区，关注反弹
```

## 三、交易体系推理层（来自向阳体系）

### 3.1 主线定位（V2）

```
判断是否为当前主线的方法：
① 全市场成交额TOP20中，同板块出现≥3只 → 可能是主线
② 板块资金连续3天净流入 → 确认
③ 龙头持续新高 → 主线还在
④ 政策/产业催化级别 → 决定主线高度

主线股特征：成交额30亿+/日、持续数周、回调有人接
非主线股特征：成交额几亿、涨一两天结束、跌了没人管
```

### 3.2 量价阶段划分（V1 + 实战经验）

将近期走势划分为清晰的量价阶段，识别主力意图：

```
阶段一：底部缩量筑底
  特征：成交量萎缩到地量、价格在低位横盘
  信号：🟢 缩量下跌 = 跌不动了 → 关注
  
阶段二：放量启动
  特征：某天突然放量+中阳/涨停
  信号：🔴 放量突破 → 第一批试探仓（V7）
  
阶段三：缩量洗盘
  特征：突破后缩量回调，不破启动点/关键支撑
  信号：🟢 缩量回调+支撑企稳 → 第二批加仓（V7）
  
阶段四：放量主升
  特征：连续放量上攻，均线多头排列
  信号：🔴 放量上涨 → 拿住别动（V10）
  
阶段五：见顶信号
  信号A：⚡ 放量滞涨 → 减仓1/3（V11）
  信号B：⚡ 量价背离（价新高量缩小）→ 分批止盈1/2（V11）
  信号C：🔴 放量跌破关键支撑 → 清仓（V11）
  
阶段六：出货完成
  特征：连续缩量阴跌或放量暴跌
  操作：🔴 回避，等下一次底部缩量
```

### 3.3 主力意图推理（结合量价）

```
强势洗盘：放量下跌 + 次日快速缩量企稳 → 主力在吓散户
温和洗盘：缩量阴跌2-3天到支撑位 → 正常技术回调
出货：放量滞涨 + 高位十字星/长上影 → 主力在卖
建仓：底部放量 + 价格稳步抬高 → 主力在买
```

### 3.4 三批建仓法（V7）

```
单票仓位上限 = 30%

第一批（试探仓）10% → 缩量回调到支撑企稳
  止损：入场价的 -8%~-10%
  
第二批（确认仓）10% → 第一批盈利 + 放量突破小压力
  如果第一批亏了 → 绝不加仓

第三批（突破仓）10% → 放量突破关键压力位
  趋势完全确认，吃主升浪
```

### 3.5 拿住别动四条件（V10）

```
四个条件全部满足 → 拿住别动
任何一条不满足 → 开始警惕

□ 价格在MA5/MA10上方（趋势向上）
□ 上涨有量配合（资金在进）
□ 回调缩量（主力没跑）
□ 没跌破关键支撑（趋势完好）
```

### 3.6 止盈三信号（V11）

| 顺序 | 信号 | 现象 | 操作 | 剩余仓位 |
|------|------|------|------|---------|
| 🥇 | **放量滞涨** | 天量+小阳线/十字星 | 减仓1/3 | 20% |
| 🥈 | **量价背离** | 新高+量<前一波70% | 分批止盈1/2 | 10% |
| 🥉 | **放量破位** | 大阴线+破MA20/前低 | 清仓 | 0% |

### 3.7 仓位管理总表（V7 + DSA风控）

| 市况 | 特征 | 总仓位 | 股票数 |
|------|------|--------|-------|
| 🟢 强势 | 主线清晰+放量上攻 | 70-90% | 3-5只 |
| 🟡 震荡 | 主线模糊+缩量震荡 | 30-50% | 2-3只 |
| 🔴 弱势 | 放量下跌+无主线 | 10-20% | 1-2只 |
| 🔥 极端 | 连续暴跌 | ≤10%或空仓 | 0-1只 |

### 3.8 空仓纪律（V6）

```
四种必须空仓的情况：
🔴 主线消失 → 龙头见顶，板块无领头羊
🔴 连续亏钱 → 连续3笔止损，节奏乱了
🔴 大盘破位 → 指数放量跌破关键支撑
🔴 情绪失控 → 开始报复性交易/追高/抄底
```

## 四、综合评分体系（DSA框架 + 向阳权重）

### 评分维度与权重

| 维度 | 权重 | 评分依据 | 来源 |
|------|------|---------|------|
| **趋势** | 30分 | 均线排列、MACD状态、趋势强度 | DSA技术指标 |
| **量价** | 25分 | 量价配合信号、量比位置、洗盘or出货 | 向阳V1体系 |
| **主线** | 20分 | 是否在资金主线、板块热度 | 向阳V2 |
| **位置** | 15分 | 乖离率、支撑/压力位、阶段位置 | DSA+向阳 |
| **风险** | 10分 | 筹码结构、新闻舆情、风险警报 | DSA风控 |

### 评分 → 操作建议映射

```
综合评分 ≥ 80 → 🟢 强烈关注/买入（多个维度共振）
综合评分 65-79 → 🟢 可以参与（条件满足）
综合评分 50-64 → 🟡 观望等待（条件不充分）
综合评分 35-49 → 🔴 谨慎/减仓（风险信号）
综合评分 < 35  → 🔴 回避/清仓（趋势破坏）
```

## 六、融合分析流程

### 第一步：查资料（DSA多数据源）
- 获取行情/日K线/技术指标
- 所属板块/筹码/新闻（如可用）

### 第二步：量化分析（DSA引擎）
- 计算均线排列、乖离率、MACD、RSI
- 量比和量价配合状态
- 综合评分（四维度加权）

### 第三步：交易体系推理（向阳体系）
- 判断所属量价阶段（V1）
- 判断是否在主线（V2）
- 检查止盈/止损信号（V11）
- 匹配当前仓位（V7）

### 第四步：融合判断
```
DSA说"评分65" + 向阳说"量价健康+在主线" → 🟢 可以参与
DSA说"评分48" + 向阳说"放量滞涨" → 🔴 回避
DSA说"评分72" + 向阳说"乖离率>5%、不在主线" → 🟡 等回调到支撑
```

### 第五步：推演+建议
- 三情景推演（向阳体系）
- 三批建仓计划（V7）
- 三止盈信号监控（V11）
- 强者思维总结（V12）
