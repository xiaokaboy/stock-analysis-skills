# 下篇：实战交易系统

## 一、交易基本功（V0）

### K线/均线/量价关系（来自193分钟基础教程 + 5/6硬核量价）

**核心方程：**
```
量 = 资金的态度（原因）
价 = 走势的结果（表现）
量在价先 —— 量变了，价才会变
```

**量价四象限：**

| | 🔴 **放量** | 🟢 **缩量** |
|--|-----------|-----------|
| **📈 上涨** | 资金跑步进场，**参与** ✅ | 惜售，一致看涨，**持有** ✅ |
| **📉 下跌** | 主力出货/恐慌，**回避** ❌ | 跌不动了，抛压衰竭，**关注** 🔍 |

**关键量价信号（按重要性排序）：**

| 信号 | 含义 | 操作 | 对应视频 |
|------|------|------|---------|
| **放量突破** | 主力正式启动 | 🟢 跟随买入 | 5/6 |
| **缩量涨停** | 惜售，一致看多 | 🟢 拿着别卖 | 5/13涨不停 |
| **放量滞涨** | 主力出货/分歧 | 🔴 **减仓1/3** | 5/15精准止盈 |
| **量价背离** | 做多动能衰竭 | 🟡 **分批止盈** | 5/15 |
| **缩量回调至支撑** | 洗盘，主力没走 | 🟢 **关注买入** | 5/19悄悄地进村 |
| **放量跌破支撑** | 趋势转空 | 🔴 **清仓** | 5/14保住利润 |

### 三个支撑位（看盘基本功）

```
第一支撑：MA5（短线生命线，强势股不该破）
第二支撑：MA10（波段趋势线，破了转弱）
第三支撑：MA20/前低（中周期趋势，不破趋势在）
```

## 二、选股：找到钱在哪里（V2·钱都在主线里，5/22+5/25）

### 核心理念
> "宁可在大主线里亏，不在小杂毛里赚"

### 四步找到主线（实战步骤）

```
第一步：收盘后看全市场成交额排名TOP20
  → 同板块出现3只以上 → 可能是主线

第二步：看板块资金流向（连续3天净流入 → 确认）
  → 同花顺/东方财富的板块资金排名

第三步：确认龙头股走势（龙头持续新高 → 主线还在）
  → 情绪看龙头，安全买跟风

第四步：验证政策/产业催化
  → 政策级别决定主线级别：工信部>行业>地方
```

### 主线 vs 非主线的区别

| 维度 | 主线股 | 非主线股 |
|------|--------|---------|
| 成交额 | 每天30亿+甚至百亿 | 几千万到几亿 |
| 持续性 | 持续数周甚至数月 | 涨一两天就结束 |
| 回调后表现 | 主力会救，缩量后继续上攻 | 跌了就是真跌 |
| 资金态度 | 跌了有人接 | 跌了没人管 |

### 案例：2026年5-6月主线演变

```
第一阶段（5月初）：稀土/有色
  龙头：北方稀土、翔鹭钨业
  逻辑：供给收缩+涨价
  结局：5/14止盈告一段落

第二阶段（5月中~6月）：光通信/AI算力 ✅ 更大主线
  龙头：工业富联、中际旭创、东山精密、烽火通信
  逻辑：AI算力爆发+800G光模块放量+工信部政策
  持续时间：贯穿整个6月，延续性极强

关键教训：
  - 第一时间发现新主线（5/6发现光通信）
  - 果断从旧主线切换（5/27~5/28调仓换股）
  - 锁定老主线利润（5/14保住利润）
```

## 三、买入：分批进场的三个条件（V7·5/6 + 6/11）

### 三个条件同时满足才出手

```
✅ 属于当前主线（V2）
✅ 缩量回调到关键支撑位（V1）
✅ 支撑位出现企稳信号（下影线/十字星/缩量）
```

### 三批建仓法（单票≤30%）

```
单票总仓位上限 = 30%（约战总资金的10-15%）

第一批（试探仓）10% → 缩量回调到支撑企稳
  → 止损设在入场价的-8%~-10%

第二批（确认仓）10% → 第一批盈利 + 放量突破小压力
  → 如果第一批亏了，绝不加仓

第三批（突破仓）10% → 放量突破关键压力位
  → 趋势完全确认，吃主升浪
```

### 实战案例：他5/19怎么进东山精密

```
① 东山精密属于光通信主线？  ✅（5/6就确认了）
② 缩量回调了吗？            ✅（从高位回踩到支撑）
③ 支撑位企稳了吗？          ✅（缩量止跌）

→ 三个条件满足 → "悄悄地进村" → 试探仓进场
→ 5/20放量上涨 → 确认加仓 → "又开始吃肉啦"
```

## 四、持仓：什么时候拿住，什么时候走（V10·6/16）

### 拿住别动的四个条件

```
价格在MA5/MA10上方 ______ ✅ 趋势向上
上涨有量配合 ______________ ✅ 资金在进
回调缩量（主力没跑）______ ✅ 洗盘特征
没跌破关键支撑 ___________ ✅ 趋势完好

四个条件全部满足 → 拿住别动
任何一条不满足 → 开始警惕
```

### 主升浪中如何拿住（从6/1想好怎么吃大肉 + 6/16拿住别动）

```
1. 主升浪途中必然有阴线洗盘 → 别被洗出去
2. 5-10%以内的回撤在上升趋势中很正常 → 别一惊一乍
3. 缩量回调是正常调整，不是见顶 → 区分洗盘和出货
4. 不要因为"赚够了"就想卖 → 趋势不坏就不卖
5. 最贵的教训不是买高了，而是卖早了——主升浪里少赚50%
```

## 五、卖出：保住利润的三个信号（V11·5/14+5/15）

### 三个止盈信号（按出现顺序）

| 顺序 | 信号 | 现象 | 操作 | 对应视频 |
|------|------|------|------|---------|
| 🥇 **放量滞涨** | 量巨大但价格涨不动 | 天量+小阳线/十字星 | 减仓**1/3** | 5/15 |
| 🥈 **量价背离** | 价格创新高但量缩小 | 新高+量<前一波70% | 分批止盈**1/2** | 5/15 |
| 🥉 **放量破位** | 放量跌破关键支撑 | 大阴线+破MA20/前低 | **清仓** | 5/14 |

### 分批止盈实操

```
总仓位30%（3批各10%）

第一批出（10%）→ 放量滞涨 → 锁定部分利润
  ↓ 如果继续涨，还有20%仓位享受
第二批出（10%）→ 量价背离 → 继续锁定
  ↓ 剩10%搏最后的鱼尾行情
第三批出（10%）→ 放量破位或趋势线 → 全部走完
  ↓ 如果前两个信号都出现了但没跌反涨 → 说明判断错误，认错
```

### 保住利润的心态（来自5/14保住利润告一段落）

> "很多人亏钱不是因为买错了，是因为**赚了没走，又跌回去了**"
> "到手的利润才是利润，账面上的不是"
> "卖了还涨不可怕，可怕的是没卖然后跌回成本"

## 六、空仓：股市生存的必修课（V6·6/5）

### 四种必须空仓的情况

```
🔴 主线消失 → 龙头见顶，板块无领头羊
🔴 连续亏钱 → 连续3笔以上止损，节奏乱了
🔴 大盘破位 → 指数放量跌破关键支撑
🔴 情绪失控 → 开始报复性交易/追高/抄底
```

### 空仓时做什么（来自V9·提升自己，6/15）

```
① 复盘最近的操作
   ✓ 哪里做对了？哪里做错了？
   ✓ 当时为什么买？信号对吗？
   
② 筛选下一波机会
   ✓ 建立观察列表
   ✓ 但不买，只跟踪
   
③ 等明确的进场信号
   ✓ 等主线重新清晰
   ✓ 等量价买入信号出现
   ✓ 你不急，市场会给你机会
```

### 不要慌不要急（来自6/10）

| 场景 | 心态 | 行动 |
|------|------|------|
| **持仓时跌了** | 在里面不要慌 | 分清是洗盘还是出货——看量！ |
| **空仓时涨了** | 在外面不要急 | 没到买点就不买，宁可错过不做错 |
| **别人赚了** | 不眼红 | 每个人节奏不同，你会有你的行情 |

```
✅ 调仓节奏（5/27→5/28→5/29）用了3天完成切换
```

## 九、新主线识别（V8·6/12）

### 主线切换的征兆

| 旧主线见顶 | 新主线启动 |
|-----------|-----------|
| 龙头放量滞涨 | 新板块出现放量突破股 |
| 板块内所有票都涨了一遍 | 新板块成交额持续放大 |
| 跟风股先于龙头跌 | 新板块有政策/产业催化 |
| 板块资金开始净流出 | 资金从旧主线流入新板块 |

### 抓住新主线的方法

```
第一步：第一时间发现
  → 每天看成交额排名前20，出现新面孔重点关注

第二步：确认强度
  → 板块内至少3只票同时走强，持续3天以上
  → 板块指数放量突破

第三步：选龙头
  → 最先涨停/涨幅最大的就是龙头
  → 做龙头还是跟风？→ 选最好的干（6/4）

第四步：找买点
  → 等龙头第一次缩量回调再进
  → 不要追高（6/2·光大涨要不要参与）
```

## 十一、强者思维（V12·6/18）

### 弱者 vs 强者

| 场景 | 弱者思维 ❌ | 强者思维 ✅ |
|------|-----------|-----------|
| 看到弹幕说跌 | 恐慌跟着卖 | 看量价自己判断 |
| 到了止盈点 | 再等等，说不定还能涨 | 按纪律止盈 |
| 亏了 | 死扛等回本 | 承认错误，止损走人 |
| 别人赚了 | 追高别人赚了的票 | 和自己比，管好自己的节奏 |
| 连续亏 | 想一把梭回来 | 停下来复盘，等状态恢复 |

### 强者思维实战应用

```
别人恐慌时 → 冷静看量价：放量出货还是缩量洗盘？
  缩量洗盘 → 机会
  放量出货 → 回避

别人狂热时 → 冷静看量价：放量突破还是放量滞涨？
  放量滞涨 → 警惕
  健康放量 → 持有
```

### 核心心法

```
赚钱是因为你遵守了规则，不是因为你聪明
亏钱是因为你违反了规则，不是因为市场坏
市场永远在波动，但你的心不能波动
用规则代替情绪，用系统代替感觉
```

# 分析流程

当用户输入**股票名称/代码**时，完整执行以下步骤：

- 股票代码、最新价、近1-2个月每日涨跌+成交量
- 所属板块 → 判断是否属于当前主线（V2）

## 第二步：量价阶段划分（核心）

把近期走势切成清晰阶段，每阶段标注信号：

```

阶段二（启动）：放量突破 → 第一批试探仓（V7）
阶段三（洗盘）：缩量回调不破支撑 → 第二批加仓（V7）
阶段四（主升）：放量上攻 → 拿住别动（V10）
阶段五（见顶）：放量滞涨/量价背离 → 按信号止盈（V11）
阶段六（出货）：放量跌破支撑 → 清仓（V11）
```

每个阶段标注：🟢健康/⚠️预警/🔴危险，并引用对应视频

## 第三步：关键检查清单

